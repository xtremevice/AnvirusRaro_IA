# Build Instructions for AnvirusRaro_IA

This document provides simple one-command build instructions for the AnvirusRaro_IA project across different operating systems.

## Windows

To build the project on Windows, run the following command in your terminal:

```bash
build.bat
```

## Linux

To build the project on Linux, execute the command below in your terminal:

```bash
./build.sh
```

## macOS

For macOS, the build command is the same as on Linux. Run:

```bash
./build.sh
```

Make sure you have all the necessary dependencies installed before running the build commands.
